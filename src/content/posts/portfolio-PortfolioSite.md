# 概要
- 個人開発
- 学生エンジニアとしての7年間の活動をまとめたポートフォリオサイト
- 実績＋ポートフォリオの管理のために作成した


# 使用技術
- Astro
  - テーマ：[Fuwari](https://astro.build/themes/details/fuwari/)
- Versel

# 機能面
Fuwariのデフォルトの機能が使えます。
- ダーク/ライトモードの切り替え
- カラーパレットの任意設定
- 記事の検索機能

# 開発のポイント
- Fuwariのテーマを残しつつ、ポートフォリオサイトとして使えるようにした点
  - Zennの記事に詳細があります。
